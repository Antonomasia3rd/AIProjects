# RealTimeNotesDeskband

Experimental/drop-in workspace for real-time notes deskband research.

This folder currently contains external/upstream-style source trees rather than a clean first-class project layout. The contents are intentionally ignored by git except for this README to avoid committing vendored dumps or unclear generated files.

## Current Contents

- `ExplorerPatcher-master`: ExplorerPatcher source snapshot/drop-in dependency area.
- `genshin-real-time-notes-0.0.8`: Genshin real-time notes source snapshot including deskband-related files.

## Status

No supported build or release flow is documented for this folder yet. Treat it as a local research workspace until the project is split into a clean source layout with explicit build steps.
