# RealTimeNotesDeskband

Native taskbar Deskband shell extension for the existing Real-Time Notes app.

## What it does

- Registers a real taskbar toolbar named `Real Time Notes`.
- Reads the existing cookie/config files:
  - `genshin_cookie.json` for Resin
  - `hsr_cookie.json` for Stamina
  - `zzz_cookie.json` for Charge
- Renders an ExplorerPatcher-weather-style compact widget:
  - icon on the left
  - `current/max` on the first line
  - time until full on the second line
- Right-click the band to refresh or switch between Resin, Stamina, and Charge.

## Requirements

This is a classic Deskband. It needs a taskbar implementation that still exposes the taskbar `Toolbars` menu, such as the Windows 10 taskbar or ExplorerPatcher's Windows 10 taskbar mode. The stock Windows 11 taskbar does not expose classic taskbar toolbars.

Build requirements:

- `g++` from MinGW-w64 or Strawberry Perl's bundled MinGW toolchain
- Existing Real-Time Notes cookie files generated by running `resin.exe`, `stamina.exe`, or `charge.exe`

## Build

From the app root:

```cmd
make deskband
```

Or run the compiler directly:

```cmd
g++ -std=c++17 -O2 -shared -static -static-libgcc -static-libstdc++ -o RealTimeNotesDeskband.dll deskband\RealTimeNotesDeskband.cpp -lole32 -loleaut32 -luuid -lshlwapi -lwinhttp -lcomctl32 -luxtheme -lgdi32 -ladvapi32
```

## Register

From the app root:

```cmd
deskband\register-deskband.cmd
```

Then right-click the taskbar, open `Toolbars`, and select `Real Time Notes`. The toolbar list may need to be opened twice after first registration.

## Unregister

```cmd
deskband\unregister-deskband.cmd
```

## Settings

Settings are stored under:

```text
HKCU\Software\RealTimeNotesDeskband
```

Values:

- `Resource`:
  - `auto`
  - `resin`
  - `stamina`
  - `charge`
- `RefreshIntervalSeconds`: optional DWORD override. The Deskband clamps values below 30 seconds.
- `InstallDir`: app root used for config files and icon assets. The registration script/DLL sets this to the DLL directory.
